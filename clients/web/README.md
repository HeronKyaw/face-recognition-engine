# Web Client

Next.js dashboard for user management, face enrollment, and verification.

Requires the backend API to be running — see [docs/setup.md](../../docs/setup.md) for the full stack setup.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.

The API URL defaults to `http://localhost:5050`. Configure via `NEXT_PUBLIC_API_URL` if different.
